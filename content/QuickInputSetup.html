                Quickly Setup for Enhanced Input

                # Introduction

                Many online tutorials use the old deprecated Axis and Action mappings system which has been replaced by the Enhanced Input system in Unreal Engine 5.0.

                The sections below show how to setup a minimal enhanced input configuration, specifically for driving a vehicle so that when we press W the vehicle moves forward.

                # Project Configuration

                Use Edit | Project Settings ... | Input to check the default input classes are as shown here:

                # Add an Input Action

                If it does not exist create a content folder called "Input/Actions"

                In that folder right-click and create an Input Action:

                Call it IA_Accelerate

                Enter a description and change the Value Type to Axis1d(float):

                Save the input action.

                # Associating the Input Action with user input

                So far we have the input action which we will later use in a blueprint, but we have not yet associated the user pressing some key with the input action. This is done using an Input Mapping Context.

                Navigate to the "Input" folder and right-click and create an Input Mapping Context:

                Call it IMC_Drive

                Open it and press the + button in the Mappings section to add a new input mapping, and set it up like this to associated the W key with the IA_Accelerate action:
